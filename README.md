# WorkFlow Pro – QA Automation Challenge

This repository contains a QA automation solution for a B2B SaaS platform.
It demonstrates test planning, UI automation, API testing, and integration testing
for a multi-tenant environment.

## Tech Stack
- Python
- pytest
- Playwright
- Requests
- BrowserStack (conceptual)

## Repository Structure
- test-plan/ – Test plan and strategy
- automation/tests/ui – UI automation tests
- automation/tests/api – API tests
- automation/tests/integration – End-to-end tests
- automation/config – Test data and environment configs

## Notes
This repository focuses on test design, scalability, and automation best practices.
